FACT Licence

Your API Terms are FAir, Transparent and Trustworthy. It’s a FACT.

O INTRO

{{user}}, The FACT (for FAIR API Commitment to Trust) license is a configurable Contract that builds on and extends traditional API ToS (Terms of Service ) to foster the use of APIs and to allow the API Users to design and exploit API Users interoperable products (whether new APIs or other types of products).

It should allow Core API Provider and API Users to reduce transaction costs Using a set of standardised contracts, as commonly accepted contracts which are considered safe by a large community. It should raise the users’ awareness of key licensing issues that the parties need to address. Furthermore, it should help companies to define a clear, unambiguous and legally consolidated framework that is favourable to the development of products interoperable with their technologies help to design high-value products and services for healthy, trustworthy and safe digital infrastructure ecosystem.

{{q1}}

The Contract is drafted neutrally to cover all rights regarding the use of APIs including all Intellectual Property rights (including patents, copyrights and trade secrets) without consideration of the validity of such rights. The standard terms of the FACT complemented by all other specific binding documents listed in a summary as “API contractual commitments summary”, constitute the entire contract.
